Rideshare Explorer
==================

A demo app visualizing a public rideshare dataset of ~900K trips in the city of Austin, TX from May 2016 to Feb 2017.

* Frontend_: React with Vega-lite, D3, Leaflet
* Server_: Python Flask
* Database_: PostgreSQL
